## Critical Constraint Clusters

### Cluster A: Determinism & Sorting (ADR 0005)
- C1.1, C3.2, C8.4, C9.3
- **Risk:** Non-deterministic output → noisy diffs → Git pollution

### Cluster B: Projection Isolation (ADR 0079)
- C1.7, C2.6, C3.1, C3.5, C4.2
- **Risk:** Direct compiled_json access → tight coupling → hard to refactor

### Cluster C: Icon Contract (ADR 0027)
- C1.2-C1.6, C5.1-C5.4, C6.3
- **Risk:** Icon resolution failure → rendering errors → user confusion

### Cluster D: ID Sanitization
- C6.2, C7.1, C7.2
- **Risk:** Invalid Mermaid IDs → graph parse errors → blank diagrams

### Cluster E: Service Dependencies
- C8.1-C8.4
- **Risk:** Dependency extraction failure → incomplete graph → misleading docs
