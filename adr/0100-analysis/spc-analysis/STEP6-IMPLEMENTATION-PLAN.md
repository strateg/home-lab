### Wave 4: Create Unified Topology Generator (REQ-NEW, S4.A)
**Goal:** New generator plugin for unified graph

**Tasks:**
1. **W4.T1:** Create `topology_graph_generator.py` plugin
2. **W4.T2:** Plugin manifest with proper dependencies:
   ```yaml
   - id: base.generator.topology_graph
     stage: generate
     execution_mode: main_interpreter  # needs ctx.config access
     depends_on:
       - base.compiler.manifest
     consumes:
       from_plugin: base.compiler.manifest
       keys:
         - compiled_json
   ```
3. **W4.T3:** Create template `unified-topology.md.j2`
4. **W4.T4:** Implement filtering logic in template:
   - Domain filter: `{% if domain_filter is none or node.domain in domain_filter %}`
   - Layer filter: `{% if layer_filter is none or node.layer in layer_filter %}`
5. **W4.T5:** Add generator config for filter defaults
6. **W4.T6:** Integration test

**Template Structure (draft):**
```jinja2
# Unified Topology Graph

Generated by `base.generator.topology_graph`.

{% set domain_filter = config.get('domain_filter') %}
{% set layer_filter = config.get('layer_filter') %}

## Filters Applied
- Domains: {{ domain_filter | join(', ') if domain_filter else 'All' }}
- Layers: {{ layer_filter | join(', ') if layer_filter else 'All' }}

```mermaid
graph TB
{% for node in topology.nodes.devices %}
{% if not domain_filter or 'physical' in domain_filter %}
{% if not layer_filter or node.layer in layer_filter %}
    {{ node.safe_id }}["{{ node.instance_id }}"]
{% endif %}
{% endif %}
{% endfor %}

{% for edge in topology.edges.host_dependencies %}
    {{ edge.source_safe_id }} -->|hosted on| {{ edge.target_safe_id }}
{% endfor %}
{# ... other edge types ... #}
```
```

**Acceptance Criteria:**
- Graph renders all node types
- Graph renders all edge types
- Filtering works (domain + layer)
- Output is deterministic
