## 🎯 Новые правила (теперь установлены)

### ПРАВИЛО 1: Analysis → `adr/00XX-analysis/`
```
Все документы анализа для ADR идут в подпапку
adr/0063-analysis/  ← Анализ для ADR 0063
adr/0064-analysis/  ← Анализ для ADR 0064 (existing)
```

### ПРАВИЛО 2: Developer guides → `docs/`
```
Все руководства для разработчиков в docs/
docs/PLUGIN_AUTHORING_GUIDE.md
docs/PLUGIN_IMPLEMENTATION_EXAMPLES.md
```

### ПРАВИЛО 3: Main ADR → `adr/00XX-*.md`
```
Основной документ ADR в корне adr/
adr/0063-plugin-microkernel-for-...md
adr/0064-os-taxonomy-...md
```
