## ✅ ЧТО НОВОГО В ADR 0064

**Было:** Property-based модель (OS как embedded property)  
**Стало:** Class-based модель (OS как first-class entity)

### Ключевые Изменения

```yaml
# Было (property-based)
device:
  software:
    os: {family, distribution, release, ...}

# Стало (class-based)
device:
  bindings:
    os: obj.os.debian.12.generic
```

### Получается

✅ No OS duplication (single definition)  
✅ Firmware vs installable distinction (explicit)  
✅ Compile-time validation (early error detection)  
✅ Multi-OS support (native)  
✅ Specialization via inheritance (clean)
