## 🔥 3 дела на ЭТОЙ НЕДЕЛЕ

**Мин. 1 час работы, макс. влияние:**

```
[ ] ДЕ № 1: Создать pyproject.toml
    ├─ Откройи IMPLEMENTATION_GUIDE.md раздел 1
    ├─ Скопируй весь блок
    ├─ Создай файл topology-tools/pyproject.toml
    └─ Запусти: pip install -e .[dev]

[ ] ДЕ № 2: Добавить 5 unit-тестов
    ├─ Откройи IMPLEMENTATION_GUIDE.md раздел 3
    ├─ Скопируй структуру тестов
    ├─ Создай tests/unit/validators/test_storage.py
    └─ Запусти: pytest tests/unit -v

[ ] ДЕ № 3: Включить mypy
    ├─ Из pyproject.toml уже есть [tool.mypy]
    ├─ Запусти: mypy topology-tools/ --ignore-missing-imports
    ├─ Исправь первые 3 ошибки типов
    └─ Видишь что работает? Успешно!
```

**Результат:** В конце недели у тебя есть базовая инфраструктура!
