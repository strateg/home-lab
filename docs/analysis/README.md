## Methodology

Documents in this directory follow the **SPC (Strict Process Compliance)** methodology:

1. **Step 0:** Read all materials
2. **Step 1:** Document Map
3. **Step 2:** Constraints Register
4. **Step 3:** Diagnostic Analysis (facts only)
5. **Step 4:** Problem Classification
6. **Step 5:** Admissible Solution Space
7. **Step 6:** Model Rebuild (approved solutions)
8. **Step 7:** Validation & Compliance Matrix

See `docs/ai/spc-contract.md` for full methodology specification.
