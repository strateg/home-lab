source_id — это идентификатор источника происхождения экземпляра.
Коротко:
•
instance — текущий ID экземпляра в v5 (уникальный ключ строки).
•
source_id — откуда он пришёл (обычно legacy/v4 ID или исходный логический ID).
Зачем нужен source_id:
•
Трассировка миграции v4 → v5.
•
Сравнение покрытий/диффов миграции.
•
Сохранение связи с исходной сущностью, даже если instance в v5 переименован.
Важно:
•
source_id может повторяться (например, один svc-grafana в v4 разделён на несколько v5-инстансов по runtime).
•
Если source_id не задан, компилятор подставляет instance (fallback).
